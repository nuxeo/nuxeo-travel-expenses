# Nuxeo Travel Expenses Sample

This sample demonstrates usage of the Workflow REST API.

## Building
### Requirements
Building this sample requires the following software:
- [Node.js] (http://nodejs.org)
- [Bower] (http://bower.io)

### How to build
Build the sample add-on with Maven:
```
mvn clean install
```
